Time: $O\((|V| + |E|)\log{|V|}\)$, \quad Space: $O\(|V|+|E|\)$

This algorithm is an improvement over SPFA (which is in turn an improvement
over Bellman-Ford) which uses a priority queue to select the next node to be
explored. It expands the search frontier from the starting node outwards until
it no longer can make any updates.
